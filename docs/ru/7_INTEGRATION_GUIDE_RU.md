## 1. СЕМАНТИЧЕСКАЯ РАЗМЕТКА (HTML) 🏷️
Добавьте атрибут `data-idx` к вашим HTML-элементам. Это позволит ИИ и вашему DevKit «видеть» точную координату объекта.

**Пример:**
```html
<button data-idx="1.A.5.SECURITY">Щит Безопасности</button>
```

## 2. ПОДКЛЮЧЕНИЕ DEVKIT (JS) 🍬
Выберите подходящий способ:

### Способ А: Для новичков (Script Tag)
Просто добавьте скрипт в ваш HTML. HUD активируется автоматически.
```html
<script src="./src/3_DEVKIT_TECHNICAL_CORE.js"></script>
```

### Способ Б: Для профи (ES-модули)
Используйте `import` для React, Vue или современных JS-проектов.
```javascript
import { UtyanskDevKit } from './src/3_DEVKIT_TECHNICAL_CORE.js';

const devKit = new UtyanskDevKit();
devKit.init();
```
